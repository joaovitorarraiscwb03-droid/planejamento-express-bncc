# Planejamento Express BNCC

Página de vendas estática profissional do infoproduto educacional **Planejamento Express BNCC**, otimizada para alta conversão no mercado mobile e desktop brasileiro.

## 🚀 Como Rodar Localmente

Para testar ou visualizar as páginas diretamente em sua máquina de forma simples:

### Opção 1 (Direto)
Basta clicar duas vezes sobre o arquivo `index.html` para abri-lo em qualquer navegador de internet.

### Opção 2 (VS Code Live Server)
1. Instale a extensão **Live Server** no VS Code.
2. Abra a pasta do projeto no editor.
3. No canto inferior direito, clique em **Go Live**.

---

## 📦 Como Subir no GitHub

Siga este passo a passo usando o terminal de sua máquina para salvar o código em um repositório remoto:

```bash
# Inicializar o repositório
git init

# Adicionar arquivos ao controle
git add .

# Criar o primeiro commit
git commit -m "primeiro deploy planejamento express bncc"

# Renomear a branch padrão
git branch -M main

# Vincular ao repositório remoto do seu GitHub
git remote add origin URL_DO_SEU_REPOSITORIO_AQUI

# Enviar os dados
git push -u origin main
